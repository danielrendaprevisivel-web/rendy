## Phase 1: Módulos Estrutura

**Goal:** Criar 3 módulos clicáveis substituindo sidebar

**Requirements:** MODULOS-01, MODULOS-02

2. Cada módulo abre área de conteúdo editável
3. Conteúdo salvo em localStorage
4. Layout funcional (cards ou abas)
